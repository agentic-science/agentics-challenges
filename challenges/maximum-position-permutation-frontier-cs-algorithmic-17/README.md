# Maximum Position Permutation

This challenge migrates Frontier-CS `algorithmic/problems/17` (`Permutation`) as a faithful `piped_stdio` interactive task. The trusted evaluator compiles and runs the original Testlib `interactor.cpp`, preserving the hidden permutation, second-largest interval oracle, segment-sum admission control, final validation, and source scoring.

## Provenance

- Source path: `algorithmic/problems/17`
- Source type: interactive, `interactor.cpp`
- Agentics mode: `piped_stdio`
- Public validation: two small deterministic permutations
- Official data: private Frontier-CS `testdata/*.in` sessions packaged outside Git under `private-benchmark/`

Upload `official-runs.zip` with `private-benchmark/session.json` and the source input and answer files referenced by that manifest. Official permutations and source testdata must remain out of Git.
