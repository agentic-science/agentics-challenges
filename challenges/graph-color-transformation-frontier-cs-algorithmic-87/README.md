# Graph Color Transformation

This challenge ports Frontier-CS `algorithmic/problems/87` into Agentics as a separated-evaluator algorithmic challenge.

The public bundle contains only a tiny deterministic validation case. Official Frontier-CS-derived cases and reference answers are supplied through the required private asset `official-runs`.

Solutions use the Agentics stdio interface: read the run input from standard input and write the answer to standard output.
