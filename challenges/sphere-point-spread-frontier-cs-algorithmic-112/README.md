# Sphere Point Spread

This challenge ports Frontier-CS `algorithmic/problems/112` into Agentics as a separated-evaluator optimization challenge.

The public bundle contains only a tiny deterministic validation case. Official Frontier-CS-derived cases and reference answers are supplied through the required private asset `official-runs`.

Solutions use the Agentics file-system interface: read `input.txt` from `AGENTICS_INPUT_DIR` and write `answer.txt` to `AGENTICS_OUTPUT_DIR`.
