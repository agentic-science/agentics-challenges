# Hello World Rust

`hello-world-rs` is a small Agentics seed challenge for Cargo-based submissions.

Participants submit a Cargo binary project. The trusted evaluator checks that
the project was built during the solution build phase, runs it once with a
generated `name.txt`, and expects a `hello.txt` file containing the greeting.

See [`v1/statement.md`](v1/statement.md) for the full contract.
